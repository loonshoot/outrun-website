Feature

Multi-tenant Isolation

Every Outrun workspace gets its own database. Your data is never mixed with other customers - it is completely separate by design.

View all features →

Capabilities

Your data, your database

Separate Databases

Each workspace has its own dedicated database. Your contacts, deals, workflows, and synced data live in a database that only your workspace can access.

Built-in Separation

Data isolation is not an add-on - it is how Outrun is built. Every workspace is separated at the database level from day one.

Encrypted Credentials

Each workspace maintains its own encrypted authentication tokens for connected tools. Your credentials are never shared across workspaces.

How isolation works

One Workspace, One Database

When you create a workspace, Outrun provisions a dedicated database for it. All your data goes there and nowhere else.

No Shared Tables

Unlike many SaaS tools that store everyone in the same tables, your data lives in its own database entirely.

Access Controls

Every request is authenticated and scoped to your workspace. There is no way to accidentally query another workspace's data.

Dedicated Infrastructure on Request

For businesses with stricter isolation requirements, we can accommodate dedicated infrastructure. Talk to our team.

Simple, honest data separation

Many platforms claim data isolation but store everything in shared tables with row-level filters. Outrun takes a different approach - each workspace gets its own database, so your data is structurally separate from other customers.
